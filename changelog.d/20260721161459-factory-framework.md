Added an async-aware test-data `Factory` framework under `velocious/src/testing/factory` (default singleton plus `createFactoryRegistry()` for isolated registries). It supports named factories, literal/lazy/dependent attributes, transient attributes, traits (global, inherited factory-local, composed) and aliases, graph-first associations wired through relationship metadata and persisted by native root autosave, numeric sequences with deterministic concurrent allocation, `build`/`create`/`attributesFor` strategies plus list/pair helpers, lifecycle callbacks with guaranteed `afterAll` cleanup, validated `initializeWith`/`toCreate`/`skipCreate`, full `modify`/`reset`, aggregated `lint`, debug events, and an opt-in Node-only definition loader (`node/load-definitions.js`) kept out of the browser-safe core. List helpers reuse one compiled declaration plan while retaining isolated per-entry evaluation and lifecycle events. See `docs/factories.md`.
