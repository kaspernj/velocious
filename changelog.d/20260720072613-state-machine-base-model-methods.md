Base-model generation (`generate:base-models`) now emits typed JSDoc stubs for a model's state-machine event methods (`event()`, `${event}AndSave()`, `can${Event}()`, `can${Event}Async()`) into the base model, so consumers no longer hand-write typecheck stubs on the model. Event names that aren't valid JavaScript identifiers are skipped with a warning. Also typed the state-machine introspection API (`getStateMachineDefinition`, `getStateMachineColumn`, `getStateMachineStateNames`) on the Record base class.
