- Restrict `_convertValue` JSON-encoding to plain objects and arrays. The previous change JSON-encoded every non-null object, which threw `Converting circular structure to JSON` when a class instance (e.g. a model record, circular via `_changes`) reached the driver as a column value — crashing the process. Class instances and Buffers now pass through untouched, as before; only plain objects/arrays (real JSON-column values) are encoded.
