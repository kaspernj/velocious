Extracted `StructureSqlLoader` (`velocious/build/src/database/structure-sql-loader.js`) from the `db:schema:load` command so a structure SQL dump can be loaded into any connection programmatically via `new StructureSqlLoader().load({db, structureSql})`. It disables foreign keys around the load and uses the driver's native multi-statement `exec` in a single round-trip when available, making it a fast way to provision a tenant/test database from a structure dump instead of materializing the schema table by table.
