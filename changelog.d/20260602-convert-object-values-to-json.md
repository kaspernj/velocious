- Serialize plain object/array column values as JSON in `_convertValue` so they land in JSON/text columns correctly. Previously an object value reached the driver's escaper as-is, and mysql's `escape()` rendered it as `` `key` = value `` assignment pairs (its `SET ?` form) — producing invalid SQL in a value position and breaking inserts into JSON columns (e.g. `syncs.data`). Buffers and Dates are left untouched.
