`StructureSqlLoader` now loads a whole structure SQL dump in a single round-trip on MySQL when the connection is configured with the new opt-in `multipleStatements: true` database option, via the new `Base#execStructureScript` driver hook. Previously only SQLite's native `exec` batched; MySQL issued every statement separately. This makes provisioning a database from a structure dump (e.g. a tenant/test database) much faster on MySQL. The option is off by default so ordinary queries keep rejecting stacked statements.
